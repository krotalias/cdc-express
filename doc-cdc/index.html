public/rational.cjs

Summary.

Desconto Racional por Dentro.

Coded following CommonJS module syntax, which was the original way to package JavaScript code for Node.js.

In fact, since this script is meant to be run on the server, via nodejs (and not on the browser), it must export its functions to serve Express, which provides a robust set of features for web and mobile applications.
